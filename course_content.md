## Course Overview

**The Dream Tree Framework** by Braedon Long

A comprehensive career development program structured in 3 parts over 15 days, designed to guide users from understanding their foundation through building their future career.

**Core Philosophy:**
- Life divided into 4 categories: Work, Play, Love, Health
- Users own their data and journey
- Honesty and self-reflection are paramount
- Growth towards your "light" (values alignment)

**Five Core Mindsets (Referenced Throughout):**
1. Curiosity
2. Bias to Action
3. Reframing
4. Awareness
5. Radical Collaboration

## Part 1: Roots (Days 1-5)
**Theme:** Understanding Your Foundation

### Module 1: Work Factors 1 - Skills and Talents

**Objective:** Complete inventory of all skills with mastery assessment

**Exercise 1.1: Transferable Skills Inventory**
- Type: Structured reflection + form
- Data collected:
  - All jobs held
  - All projects completed
  - All courses taken
  - Tasks performed in each
  - Skill mastery ratings (a-e scale)
- Output: List of transferable skills with mastery grades

**Exercise 1.2: SOARED Stories**
- Type: Narrative writing (3 stories minimum)
- Prompt framework:
  - 9 prompts for challenge scenarios (overcome, impact, excited, etc.)
  - Choose 3 most impactful
  - Write using SOARED acronym:
    - Situation
    - Obstacle
    - Action
    - Result
    - Evaluation
    - Discovery
- Data collected: Detailed story narratives
- Output: Skills extracted from stories

**Exercise 1.3: Skills Ranking**
- Type: Ranking exercise
- Data collected:
  - Top 10 mastered skills
  - Ranked by importance (1-10)
- Tool: Ranking grid
- Output: Prioritized skill list

**Exercise 1.4: Self-Management Skills (Soft Skills)**
- Type: Personality assessment + reflection
- Components:
  - Myers-Briggs adaptation (16personalities.com)
  - Four-letter type code
  - Self-assessment on 4 scales:
    - E/I (Extroversion/Introversion)
    - S/N (Sensing/Intuition)
    - T/F (Thinking/Feeling)
    - J/P (Judging/Perceiving)
- Data collected:
  - Personality type
  - 10 favorite soft skills (ranked)
  - Characteristic matches/mismatches
- Output: Soft skill profile

**Exercise 1.5: Skills with Context**
- Type: Descriptive writing
- Data collected:
  - Top 10 transferable skills described with soft skills
  - Stories/examples of usage
  - Quantified accomplishments (numbers, data, records)
- Output: Professional skill descriptions

**Exercise 1.6: Knowledges**
- Type: Comprehensive list + assessment
- Data collected:
  - All jobs (skills/processes from each)
  - All courses/training/degrees (skills from each)
  - Hobbies
  - Research interests
  - Book topics they'd write
  - Time-flying activities
  - Interesting fields/careers/industries
  - Mastery grades for each knowledge
- Output: Complete knowledge inventory with mastery ratings

### Module 2: Work Factors 2 - Environment, People, Compensation

**Exercise 2.1: Macro Location (Where in the World)**
- Type: Exploration + research + ranking
- Data collected:
  - Desired location (if known)
  - Location characteristics desired
  - Location constraints
  - Places previously lived
  - Dislikes about previous places
  - Opposite traits (what they want)
  - Researched locations (5-10)
  - Top 10 possible locations (ranked)
  - **Top 3 locations flagged**
- Partner variation: Combined exercise if user has partner
- Output: Prioritized location list

**Exercise 2.2: Micro Environment (Workplace Feel)**
- Type: Reflection on past workspaces
- Data collected:
  - All jobs held
  - Positive work conditions from each
  - Negative work conditions from each
  - Opposites of negatives
  - Ideal workplace description:
    - Office appearance
    - What's nearby
    - Important features
    - City/town location
    - Proximity to childcare
    - Other considerations
- Output: Ideal workplace profile

**Exercise 2.3: People**
- Type: Reflection + trait analysis
- Data collected:
  - Harmful traits of past colleagues (what hurt productivity)
  - Antonyms of harmful traits
  - Helpful traits of past colleagues
  - Traits from highly productive periods
  - Supportive traits for work environment
  - Preference: teams/pairs/alone
  - Types of people want to work with
  - Types of people want to work for
  - Who they want to serve with their work
- Output: People profile

**Exercise 2.4: Compensation**
- Type: Financial planning + preference assessment
- Data collected:
  - All past positions with pay history
  - Complete budget (monthly expenses)
  - Total annual expenses
  - Minimum hourly wage needed
  - Required benefits
  - Compensation type preferences:
    - Salary (pros/cons)
    - Hourly (pros/cons)
    - Contracting (pros/cons)
    - Commission (pros/cons)
    - Combination/Other (pros/cons)
- Output: Compensation requirements and preferences

### Module 3: Priorities, Ecstasy & Flow

**Exercise 3.1: Work Factors Priority Ranking**
- Type: Ranking exercise
- Data collected:
  - 5 work factors ranked:
    1. Compensation
    2. Environment
    3. Skills
    4. Knowledges
    5. People
  - Optional: More specific sub-priorities
- Tool: Ranking grid
- Output: Prioritized work factors

**Exercise 3.2: Flow Tracking - Memorable Past**
- Type: Retrospective assessment
- Data collected:
  - All significant events from Days 1-2
  - Energy rating for each (-2 to +2: draining to energizing)
  - Focus rating for each (v to z: boring to captivating)
  - Flow moments flagged
  - Top 5-10 most focused activities
  - Top 5-10 most energizing activities
  - Activities that are BOTH highly energizing AND captivating
- Tool: "Tracking Flow" worksheet
- Output: Flow profile from past

**Exercise 3.3: Flow Tracking - Recent Past**
- Type: Daily tracking (past 3-10 days)
- Data collected:
  - Every significant activity per day
  - Short description of each
  - Energy and focus ratings
  - Activities in flow state
- Tool: "Tracking Flow" worksheet
- Output: Recent flow patterns

**Exercise 3.4: Flow Tracking - Present (Ongoing)**
- Type: Daily habit (continues through Week 3)
- Data collected:
  - Daily activity tracking
  - Energy/focus ratings
  - Flow identification
- Tool: "Tracking Flow" worksheet
- Output: Evolving flow understanding

- Type: Summary + rating
- Data collected:
  - Work summary + rating (0-10)

  - Non-monetary gains from work

- Data collected:

- Type: Synthesis analysis

### Module 8: Landmarking

**Exercise 8.1: New Growth (Idea Trees)**
- Type: Creative brainstorming
- Process:
  - Choose 3 topics that energize AND captivate (from flow tracking)
  - Create 3 idea trees:
    - Trunk = starter word
    - 3-4 branches from trunk
    - 3-4 branches from each branch (layer 2)
    - 3-4 branches from each layer 2 branch (layer 3)
    - Optional: branches from layer 3 if needed
  - NO CENSORING IDEAS
- Data collected:
  - 3 complete idea trees
  - All branching ideas
- Output: Idea generation practice

**Exercise 8.2: Reading the Leaves**
- Type: Job ideation
- Process:
  - Pick 3 ideas from outermost branches of each tree
  - Brainstorm jobs combining those 3 ideas
  - Repeat for all 3 trees
- Data collected:
  - Job ideas combining branched concepts
- Output: Creative career possibilities

**Exercise 8.3: Making Choices (3 Career Options)**
- Type: Decision-making
- Requirements:
  - Design 3 full profession options
  - NOT specific employers, but PROFESSIONS
  - Consider: skills, interests, learning potential
  - Option 1: Best fit (needs + skills + interests)
  - Option 2: Solid alternative
  - Option 3: "Why not?" option (ambitious/curious)
- Data collected:
  - 3 chosen professions with rationale
- Output: Career direction options

## Key Data Collection Points for AI Context

**Character Analysis Triggers (Managing Bot):**
- After Module 5 completion
- After Module 9 completion
- After Module 14 completion
- After significant 15+ message conversations

**Skills That Need Character Notes:**
1. Communication style (verbose vs concise, emotional vs analytical)
2. Energy patterns (what topics create expansion vs brevity)
3. Decision-making approach (deliberate vs intuitive)
4. Self-awareness level
5. Coaching needs (encouragement, challenge, structure)
6. Unique quirks (metaphors used, topics referenced)

**Critical Data for Credentials:**
- Module completion timestamps
- Exercise completion status
- Quality scores (based on depth/thoughtfulness)
- Time invested per module
- Overall course completion

**AI Tutor Context Needs:**
Throughout course, AI needs access to:
- All previous exercise responses
- Character notes from previous modules
- Flow tracking data
- Values alignment
- Career options chosen
- Target companies/roles
